== Description ==

WordPress automatically discovers enclosures you reference in a post. Use this plugin to stop WordPress from doing that.

All other enclosures will work. Existing enclosures will not be deleted.

= Quick Start =

Activate the plugin.
Enjoy :)

== Installation ==

1. Upload the `no-more-enclosures` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. There is no step 3
